Velocity from Position

E02 Foundations Numerical Derivatives Pendulum Est. 1964

Numerical differentiation amplifies noise catastrophically. See why faster sampling can make velocity estimates worse, not better.

Prerequisite: E01: Noisy Sensors
Robot View
0 / 200
Noise \(\sigma\): 0.050
Sample \(\Delta t\) (s): 0.020

Overview

Robots need velocity for control, but many sensors only measure position. The naive approach — finite-difference differentiation $\dot{q} \approx (q_{k} - q_{k-1})/\Delta t$ — amplifies measurement noise by a factor of $\sigma\sqrt{2}/\Delta t$.

Key Insight

Decreasing the sample period $\Delta t$ makes differentiation noise worse, not better. The noise amplification factor is $\sqrt{2}/\Delta t$, which grows without bound as $\Delta t \to 0$. This is the fundamental motivation for model-based observers (E03) and Kalman filters (E04).

When to Use

  • Quick velocity estimate when noise is low and sample rate is moderate
  • Baseline comparison for evaluating filter performance
  • Understanding the noise-bandwidth tradeoff in digital derivatives

Advantages

  • No model required
  • Zero latency (causal, one-step delay)
  • Trivial to implement

Limitations

  • Catastrophic noise amplification
  • Worse at higher sample rates
  • No acceleration estimate

How It Works

  1. Position measurement: $y_k = q_k + v_k$ where $v_k \sim \mathcal{N}(0, \sigma^2)$
  2. Backward difference: $\hat{\dot{q}}_k = (y_k - y_{k-1}) / \Delta t$
  3. Noise propagation: $\hat{\dot{q}}_k = \dot{q}_k + (v_k - v_{k-1}) / \Delta t$
  4. Result: Velocity noise has std $\sigma_{\dot{q}} = \sigma\sqrt{2}/\Delta t$

Try decreasing $\Delta t$ in the slider — watch the noisy velocity (red) explode while the position noise stays constant.

Key Equations

Finite Difference

$$\hat{\dot{q}}_k = \frac{y_k - y_{k-1}}{\Delta t} = \dot{q}_k + \frac{v_k - v_{k-1}}{\Delta t}$$

Velocity Noise Variance

$$\text{Var}\!\left(\frac{v_k - v_{k-1}}{\Delta t}\right) = \frac{2\sigma^2}{\Delta t^2}$$

Noise Amplification Factor

$$\text{AF} = \frac{\sigma_{\dot{q}}}{\sigma_q} = \frac{\sqrt{2}}{\Delta t}$$

MATLAB Implementation

MATLAB
%% E02: Velocity from Position
sigma = 0.05; dt = 0.02; T = 4; f = 0.5; A = 1.0;
t = 0:dt:T;
q_true = A*sin(2*pi*f*t);
qd_true = A*2*pi*f*cos(2*pi*f*t);
y = q_true + sigma*randn(size(t));

% Finite difference
qd_est = diff(y)/dt;

% Predicted noise std
vel_noise_std = sigma*sqrt(2)/dt;
fprintf('Vel noise std: predicted=%.2f\n', vel_noise_std);

figure;
subplot(2,1,1); plot(t,q_true,'g',t,y,'y.'); title('Position');
subplot(2,1,2); plot(t(2:end),qd_true(2:end),'g',t(2:end),qd_est,'r');
title(sprintf('Velocity (noise amplified %.0fx)', sqrt(2)/dt));
